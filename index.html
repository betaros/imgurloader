    Welcome to Imgurloader

    Imgurloader downloads images from Imgur albums and creates a PDF and/or a CBZ file. If none is selected it downloads the images in a folder for you.

    Usage

    For using it you need an ID and a secret from (Imgur)[https://api.imgur.com/oauth2/addclient]. When you first start the tool it will ask for it and creates an XML file with the credentials.

    The files are downloaded to the folder, where the JAR was executed.
